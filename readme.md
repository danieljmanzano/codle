# CODLE

Este é um clone do jogo de adivinhações de palavras "Termo" (ou do "Wordle", a versão em inglês), feito como um projeto de aprendizado de férias sugerido pelo CodeLab. 

## Tecnologias 

- React 19: Biblioteca para construção da interface.
- Vite 7: Ferramenta de build e servidor de desenvolvimento.
- Tailwind CSS 3.4: Framework CSS utilitário para estilização.
- PostCSS: Processador de CSS para suporte ao Tailwind.

## Como Rodar

1. Clone o repositório:
```bash
   git clone https://github.com/danieljmanzano/codle
```

2. Instale as dependências:
```bash
   npm install
```

3. Inicie o servidor
```bash
   npm run dev
```
